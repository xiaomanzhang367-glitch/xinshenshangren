# 新神上任

一个轻松休闲的模拟经营 + AI叙事互动游戏。玩家扮演刚刚上任的新神明，每天处理信徒的愿望，通过自己的选择影响他们的人生。

## 项目特点

- 🎮 **轻松休闲** - 7天的短流程，5-10分钟即可体验完整游戏
- 🙏 **愿望系统** - 每个愿望提供三个选择，消耗神力实现愿望
- 📱 **人间观察** - 朋友圈形式展示信徒们的生活点滴
- 💬 **神仙群聊** - 与其他神仙互动，获得提示和吐槽
- ✨ **动态神格** - 根据你的选择不断变化的神格系统
- 🔗 **故事线** - 多条主线和隐藏剧情等你发现
- 🎨 **赛博土地庙风格** - 独特的东方+赛博朋克视觉效果

## 技术栈

- React 18
- Vite
- 纯 CSS (赛博朋克风格)

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

然后在浏览器中打开显示的地址（通常是 http://localhost:3000）

### 构建生产版本

```bash
npm run build
```

## 游戏玩法

1. **开始游戏** - 欢迎来到你的神庙！
2. **处理愿望** - 每天会收到信徒的愿望，选择如何回应
   - 全力实现：消耗神力较多，成功率最高
   - 稍微帮忙：消耗神力较少，结果随机
   - 忽略：不消耗神力，可能造成负面影响
3. **观察人间** - 查看朋友圈动态，了解信徒的生活变化
4. **与神交流** - 在神仙工作群中与其他神仙互动
5. **成长神格** - 你的选择会影响神格属性
6. **探索故事** - 发现多条故事线和隐藏剧情

## 游戏资源

- **香火** - 通过实现愿望获得，用于衡量信徒满意度
- **神力** - 实现愿望时消耗，每天会恢复一定数值
